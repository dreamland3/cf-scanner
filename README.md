# 🚀 Cloudflare Smart Scanner Bot (cf-scanner)

یک سیستم هوشمند و دوگانه برای تست، اسکن و یافتن آی‌پی‌های تمیز و بدون فیلتر کلودفلر مخصوص اپراتورهای مختلف در ایران. این پروژه از دو بخش ربات تلگرام (بک‌اند پایتون) و مینی‌آپ/وب‌آپ (فرانت‌اند جاوااسکریپت) تشکیل شده است.

---

## ✨ قابلیت‌های کلیدی

- **اسکن زنده:** قابلیت اسکن مستقیم رنج‌های کلودفلر از داخل محیط مینی‌آپ تلگرام بدون نیاز به نصب هیچ نرم‌افزار جانبی (مخصوص کاربران موبایل و اتصال از طریق پروکسی).
- **لیست آماده اپراتورها:** استخراج و نمایش چرخشی آخرین آی‌پی‌های تمیز ثبت‌شده برای اپراتورهای مختلف از طریق لایه `DNS Resolution`.
- **پشتیبانی از اپراتورها:** همراه اول، ایرانسل، رایتل، مخابرات، شاتل، پیشگامان و مبین‌نت.
- **طراحی بهینه و سبک:** اجرای کاملاً مستقل در پس‌زمینه سرور به صورت لینوکس سرویس بدون تداخل با سایر ابزارهای شبکه.
- **امنیت بالا:** جداسازی کامل توکن‌های حساس ربات از بدنه کد گیت‌هاب و تزریق امن آن‌ها از طریق متغیرهای محیطی سیستم‌عامل (`Environment Variables`).

---

## 🛠️ ساختار فنی پروژه

این پروژه از دو بخش مجزا اما متصل به هم تشکیل شده است:

1. **بخش فرانت‌اند:** کدهای `HTML` / `JS` / `CSS` که در روت اصلی مخزن قرار دارند و از طریق **GitHub Pages** میزبانی می‌شوند تا به عنوان وب‌آپ در تلگرام لود شوند.
2. **بخش بک‌اند:** فایل `bot.py` که با کتابخانه `pyTelegramBotAPI` نوشته شده و روی سرور لینوکس در دیتاسنتر اجرا می‌شود.

---

## 🚀 راهنمای راه‌اندازی ربات روی سرور لینوکس

برای اجرای پایدار این ربات در پس‌زمینه لینوکس، مراحل زیر انجام شده است:

### ۱. ساخت سرویس اختصاصی لینوکس
یک فایل سرویس در مسیر زیر ساخته می‌شود:
`/etc/systemd/system/cf-scanner-bot.service`

پیکربندی داخل این فایل به شرح زیر است:

```ini
[Unit]
Description=Cloudflare Scanner Telegram Bot
After=network.target

[Service]
Type=simple
User=root
WorkingDirectory=/root/cf-scanner
ExecStart=/root/tg_bot_env/bin/python3 bot.py
Environment="BOT_TOKEN=YOUR_SECRET_TELEGRAM_TOKEN"
Restart=always
RestartSec=5
StandardOutput=append:/root/cf-scanner/bot.log
StandardError=append:/root/cf-scanner/bot.log

[Install]
WantedBy=multi-user.target
۲. دستورات مدیریت سرویس ربات
برای مدیریت این ربات روی سرور از دستورات استاندارد زیر استفاده می‌شود:

شروع به کار و فعال‌سازی خودکار در زمان بوت سرور:

Bash
sudo systemctl daemon-reload
sudo systemctl enable cf-scanner-bot.service
sudo systemctl start cf-scanner-bot.service
بررسی وضعیت زنده بودن ربات:

Bash
sudo systemctl status cf-scanner-bot.service
مشاهده لاگ‌ها و خطاهای احتمالی به صورت زنده:

Bash
tail -f /root/cf-scanner/bot.log
